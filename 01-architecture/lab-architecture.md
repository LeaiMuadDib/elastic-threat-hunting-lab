# Lab Architecture

This lab environment simulates a real-world security monitoring stack using Elastic Security.

Components:

Infrastructure
- Mini PC server
- Proxmox virtualization host

Security Platform
- Elasticsearch
- Kibana
- Fleet Server
- Elastic Defend

Endpoints
- Windows laptop
- macOS workstation

The environment is used to practice detection engineering, threat hunting, and incident investigations using simulated adversary techniques.
