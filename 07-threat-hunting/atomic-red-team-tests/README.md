# Atomic Red Team Testing

## Purpose

Atomic Red Team is used to simulate real attacker techniques mapped to the MITRE ATT&CK framework. These tests generate telemetry within the Elastic Security environment, allowing validation of detection rules and investigation workflows.

## Environment

Atomic tests were executed from the Windows endpoint with Elastic Agent installed and enrolled in Fleet.

## Goals

- Generate realistic attacker telemetry
- Validate Elastic prebuilt detection rules
- Practice threat hunting workflows
- Document incident investigations
